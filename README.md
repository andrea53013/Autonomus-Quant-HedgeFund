# Autonomus Quant Hedge Fund

This repository contains the code for the Autonomus Quant Hedge Fund, a quantitative trading system.

## Overview

- **src/**: Source code related to the trading strategies.
- **README.md**: Project overview and instructions.
- **INSTALL.md**: Installation instructions.
- **.gitignore**: Files and directories to ignore by git.
- **Autonomus-Quant-HedgeFund.sln**: Solution file for Visual Studio.
